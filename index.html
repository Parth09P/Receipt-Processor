Welcome to Receipt Processor API

This API allows you to process receipts and retrieve points.

1. Process Receipt

This API processes a receipt and calculates the points based on specified rules.
Endpoint: /receipts/process - POST

2. Get Points

Use this API to retrieve points for a previously processed receipt using its unique ID.
Endpoint: /receipts/{receipt_id}/points - GET

For testing the APIs, visit the FastAPI Docs.
